At the recent H2AI Hackathon hosted by Georgetown University, my team and I reached the finals by tackling a significant challenge in medical decision-making. We developed the "Info Button," an AI-powered tool designed to support healthcare professionals faced with complex medication decisions. Our solution retrieves the latest medical research and guidelines, integrating them directly into clinical practice through an interactive platform. Built using Node.js, this platform features an AI chatbot that enables doctors to maintain comprehensive patient profiles and access updated treatment options based on current best practices.

In a personal project, I developed a self-hosted music server using a Raspberry Pi, centralizing my music library for both local and remote streaming. This endeavor enhanced my understanding of embedded systems and reinforced my programming skills in Python and C++, crucial for full-stack project management and robotics applications. Additionally, I am currently developing a project in Rust utilizing the Polygon API to fetch financial data. This data is processed through the Claude API Opus, which cleans and formats the information, before generating a comprehensive LaTeX-formatted report on stocks or portfolios. This project demonstrates my ability to integrate complex APIs and produce valuable financial analytics tools.

Last summer, I developed a simple speech-to-text program leveraging Google's Cloud Services and its speech-to-text API, enhancing my experience with cloud-based technologies. I also have professional experience as a Software Engineer intern at Jackpine Technologies, a defense software company based in Maynard, MA. Here I spent most of my time translating and implementing automation tests in Java and Groovy. I also adapted scripts across Windows and Linux platforms using JSON properties. My time at Jackpine was valuable, deepening my understanding of the software development lifecycle and strengthening my desire to produce amazing software for this country.

This summer I am interning at Quikirr as a Software Engineer to help develop their Excel financial modeling tool.
